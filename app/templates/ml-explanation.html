FIFA 21 Player Position Classification — GK / DEF / MID / FWD

Dataset: FIFA 21 Messy Raw

ขนาดข้อมูล 18,979 rows × 77 columns
แหล่งที่มา Kaggle — FIFA 21
Target Position Group: GK / DEF / MID / FWD
Task Multi-class Classification (4 คลาส)
Split Train 70% / Val 15% / Test 15%
ปัญหาใน Messy Data
  • Height/Weight: มีทั้ง cm/ft และ kg/lbs ปนกัน
  • Value/Wage: format "€1.6M", "€200K" ต้องแปลงเป็น float
  • W/F, SM, IR: มีสัญลักษณ์ ★ ต้องลบออก
  • Missing Values: Release Clause, Hits, Loan Date End
  • Contract: มีทั้ง date / "Free" / "On Loan" ปนกัน

ทฤษฎี Ensemble Learning

Random Forest

สร้าง Decision Trees หลายต้นด้วย Bootstrap Sampling และ Random Feature Selection แล้วรวมผลด้วย Majority Vote

ข้อดี: ทนต่อ Overfitting · ตีความ Feature Importance ได้

n_estimators, max_depth, min_samples_split
XGBoost

Gradient Boosting ที่ optimize ด้วย Second-Order Taylor Expansion พร้อม L1/L2 Regularization

ข้อดี: ความแม่นยำสูง · Parallel Training

learning_rate, subsample, colsample_bytree
LightGBM

Gradient Boosting แบบ Leaf-wise Tree Growth ใช้ Histogram เร็วกว่า XGBoost หลายเท่า

ข้อดี: ฝึกเร็ว · Memory น้อย

num_leaves, min_data_in_leaf
Stacking

รวม 3 โมเดลข้างต้นเป็น Base Learners แล้วใช้ Logistic Regression เป็น Meta-Learner

ข้อดี: ใช้จุดแข็งของทุกโมเดล

cv=5, stack_method='predict_proba'

# แปลง Height → cm
def parse_height(h):
    if 'cm' in str(h): return float(str(h).replace('cm',''))
    elif "'" in str(h):
        parts = str(h).replace('"','').split("'")
        return round(int(parts[0])*30.48 + int(parts[1] or 0)*2.54, 1)

# แปลง Value/Wage → float
def parse_money(val):
    val = str(val).replace('€','')
    if 'M' in val: return float(val.replace('M','')) * 1_000_000
    if 'K' in val: return float(val.replace('K','')) * 1_000
    return float(val)

# ลบ ★ จาก W/F, SM, IR
for col in ['W/F','SM','IR']:
    df[col] = df[col].str.replace('★','').str.strip()
    df[col] = pd.to_numeric(df[col], errors='coerce')
