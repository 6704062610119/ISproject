Information

โมเดลนี้เกี่ยวกับการจำแนกตำแหน่งผู้เล่นฟุตบอลในเกม FIFA 21 โดยใช้ข้อมูลสถิติของผู้เล่น เพื่อทำนายว่าผู้เล่นคนนั้นอยู่ในตำแหน่ง GK (ผู้รักษาประตู), DEF (กองหลัง), MID (กองกลาง) หรือ FWD (กองหน้า) โดยใช้เทคนิค Ensemble Learning แบบ Stacking ที่รวม Random Forest, XGBoost และ LightGBM เป็น Base Learners และใช้ Logistic Regression เป็น Meta-Learner

DataSet

ขนาดข้อมูล 18,979 rows × 77 columns
แหล่งที่มา Kaggle — FIFA 21
Target Position Group: GK / DEF / MID / FWD
Task Multi-class Classification (4 คลาส)
Split Train 70% / Val 15% / Test 15%
ปัญหาใน Messy Data
  • Height/Weight: มีทั้ง cm/ft และ kg/lbs ปนกัน
  • Value/Wage: format "€1.6M", "€200K" ต้องแปลงเป็น float
  • W/F, SM, IR: มีสัญลักษณ์ ★ ต้องลบออก
  • Missing Values: Release Clause, Hits, Loan Date End
  • Contract: มีทั้ง date / "Free" / "On Loan" ปนกัน

ทฤษฎี Ensemble Learning

Random Forest

สร้าง Decision Trees หลายต้นด้วย Bootstrap Sampling และ Random Feature Selection แล้วรวมผลด้วย Majority Vote

ข้อดี: ทนต่อ Overfitting · ตีความ Feature Importance ได้

n_estimators, max_depth, min_samples_split
XGBoost

Gradient Boosting ที่ optimize ด้วย Second-Order Taylor Expansion พร้อม L1/L2 Regularization

ข้อดี: ความแม่นยำสูง · Parallel Training

learning_rate, subsample, colsample_bytree
LightGBM

Gradient Boosting แบบ Leaf-wise Tree Growth ใช้ Histogram เร็วกว่า XGBoost หลายเท่า

ข้อดี: ฝึกเร็ว · Memory น้อย

num_leaves, min_data_in_leaf
Stacking

รวม 3 โมเดลข้างต้นเป็น Base Learners แล้วใช้ Logistic Regression เป็น Meta-Learner

ข้อดี: ใช้จุดแข็งของทุกโมเดล

cv=5, stack_method='predict_proba'

Machine Learning Models

ทำการ Data Cleaning และ Data Preprocessing โดยแปลงหน่วย Height/Weight, จัดการค่า Value/Wage จากรูปแบบ €M/€K ให้เป็น float, ลบสัญลักษณ์ ★ ออกจาก W/F, SM, IR และจัดกลุ่ม Position เป็น 4 กลุ่ม 

# 1. แปลง Height → cm
def parse_height(h):
    if 'cm' in str(h): return float(str(h).replace('cm',''))
    elif "'" in str(h):
        parts = str(h).replace('"','').split("'")
        return round(int(parts[0])*30.48 + int(parts[1] or 0)*2.54, 1)

# 2. แปลง Value/Wage → float  (€1.6M → 1600000)
def parse_money(val):
    val = str(val).replace('€','')
    if 'M' in val: return float(val.replace('M','')) * 1_000_000
    if 'K' in val: return float(val.replace('K','')) * 1_000
    return float(val)

# 3. ลบ ★ จาก W/F, SM, IR
for col in ['W/F','SM','IR']:
    df[col] = df[col].str.replace('★','').str.strip()
    df[col] = pd.to_numeric(df[col], errors='coerce')

# 4. จัดกลุ่ม Position → GK / DEF / MID / FWD
position_map = {
    'GK': 'GK',
    'CB':'DEF','LB':'DEF','RB':'DEF','LWB':'DEF','RWB':'DEF',
    'CM':'MID','CDM':'MID','CAM':'MID','LM':'MID','RM':'MID',
    'ST':'FWD','CF':'FWD','LW':'FWD','RW':'FWD','LS':'FWD','RS':'FWD'
}
df['Position_Group'] = df['Best Position'].map(position_map)
df.dropna(subset=['Position_Group'], inplace=True)

# 5. กำหนด X และ y
features = ['Age','Overall','Potential','Value_EUR','Wage_EUR',
            'Height_cm','Weight_kg','Pace','Shooting','Passing',
            'Dribbling','Defending','Physic','W/F','SM','IR']
X = df[features]
y = df['Position_Group']

ทำการสร้าง Random Forest, XGBoost, LightGBM เพื่อ Stacking และ Train โมเดล 

# สร้าง Base Learners
rf  = RandomForestClassifier(n_estimators=200, max_depth=20, random_state=42)
xgb = XGBClassifier(n_estimators=300, learning_rate=0.05,
                    max_depth=6, random_state=42, eval_metric='mlogloss')
lgb = LGBMClassifier(n_estimators=300, learning_rate=0.05,
                     num_leaves=63, random_state=42)

# Stacking Classifier (Meta-Learner = Logistic Regression)
model = StackingClassifier(
    estimators=[('rf', rf), ('xgb', xgb), ('lgb', lgb)],
    final_estimator=LogisticRegression(max_iter=1000),
    cv=5,
    stack_method='predict_proba'
)

# เทรนโมเดล
model.fit(X_train, y_train)

# ประเมินผล
y_pred = model.predict(X_test)
print(f"--- 📊 ผลการทดสอบ Stacking Model ---")
print(f"Accuracy: {accuracy_score(y_test, y_pred):.4f}")
print(classification_report(y_test, y_pred))
