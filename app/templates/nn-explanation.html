Neural Network (MLP)

EPL Match Result Prediction — Home Win / Draw / Away Win

Dataset: English Premier League

ขนาดข้อมูล 380 rows (ฤดูกาล 2020–2021)
แหล่งที่มา Kaggle — EPL Matches
Target H = Home Win · D = Draw · A = Away Win
Task Multi-class Classification (3 คลาส)
Split Train 70% / Val 15% / Test 15%
ปัญหาใน Dataset
  • Betting odds: มีคอลัมน์ odds หลายสิบคอลัมน์ ต้องเลือกเฉพาะ match stats
  • HS / AS: Home/Away Shots — ครบทุกนัด
  • HST / AST: Shots on Target — ครบทุกนัด
  • HC / AC: Corners — ครบทุกนัด
  • Sample size: 380 นัด (1 ฤดูกาล) เป็นข้อมูลน้อย อาจบิดให้โมเดล generalize ได้จำกัด

ทฤษฎี MLP Neural Network

สถาปัตยกรรม

Multi-Layer Perceptron (MLP) คือ Neural Network ชนิดพื้นฐาน ประกอบด้วย:

Layer Type Details
Input — N features
Hidden 1 Dense 256 ReLU + BatchNorm + Dropout(0.3)
Hidden 2 Dense 128 ReLU + BatchNorm + Dropout(0.3)
Hidden 3 Dense 64 ReLU + BatchNorm + Dropout(0.15)
Output Dense 3 Softmax
Training Config
Optimizer Adam (lr=0.001)
Loss Categorical Crossentropy
Epochs 200 (with EarlyStopping)
Batch Size 32
EarlyStopping patience=25, monitor=val_accuracy
ReduceLROnPlateau factor=0.5, patience=10
ModelCheckpoint save_best_only=True
Batch Normalization

Normalize activations ในแต่ละ batch ทำให้ training เสถียรขึ้น เร็วขึ้น และ Gradient ไหลได้ดีขึ้น

Dropout

สุ่มปิด neurons บางส่วนระหว่างเทรน ป้องกัน Overfitting ทำให้โมเดล generalize ได้ดีขึ้น

ReLU Activation

f(x) = max(0, x) แก้ปัญหา Vanishing Gradient ทำให้ Deep Network เทรนได้มีประสิทธิภาพ

โค้ดสร้าง MLP

03_neural_network.ipynb 
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense, Dropout, BatchNormalization
from tensorflow.keras.optimizers import Adam

model = Sequential([
    Dense(256, activation='relu', input_shape=(n_features,)),
    BatchNormalization(),
    Dropout(0.3),

    Dense(128, activation='relu'),
    BatchNormalization(),
    Dropout(0.3),

    Dense(64, activation='relu'),
    BatchNormalization(),
    Dropout(0.15),

    Dense(3, activation='softmax')   # H / D / A
])

model.compile(
    optimizer=Adam(learning_rate=0.001),
    loss='categorical_crossentropy',
    metrics=['accuracy']
)

history = model.fit(
    X_train, y_train_ohe,
    validation_data=(X_val, y_val_ohe),
    epochs=200, batch_size=32,
    callbacks=[EarlyStopping(patience=25), ModelCheckpoint(...)]
)
